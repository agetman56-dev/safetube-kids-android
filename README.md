# SafeTube Kids (Android)

Дитячий застосунок-аналог YouTube Kids для планшета. Показує відео **лише** зі схваленого
батьком списку YouTube-каналів (`app/src/main/assets/whitelist.json`), без пошуку, без чужих
рекомендацій. Другий рубіж захисту — мовний фільтр (`LangFilter.kt`, портований з розширення
SafeTube UA), який ховає навіть відео зі схваленого каналу, якщо воно російськомовне.

## Перш ніж збирати APK

1. **Створи порожній репозиторій на GitHub** (без README/gitignore — вони вже тут), наприклад
   `safetube-kids-android`.
2. Постав код у нього (заміни `<URL>` на адресу свого репозиторію):
   ```
   git init
   git add .
   git commit -m "Початкова версія SafeTube Kids"
   git branch -M main
   git remote add origin <URL>
   git push -u origin main
   ```
3. **Додай секрет з ключем YouTube Data API v3**: у репозиторії на GitHub —
   Settings → Secrets and variables → Actions → New repository secret →
   назва `YOUTUBE_API_KEY`, значення — той самий ключ, що вже використовує розширення SafeTube UA.
4. Після push автоматично запуститься збірка (вкладка **Actions** у репозиторії). Через ~3-5 хв
   з'явиться artifact `safetube-kids-debug` — це готовий APK, качай і встановлюй на планшет
   (може знадобитись дозволити "Встановлення з невідомих джерел" в налаштуваннях Android).

## Список каналів (whitelist.json)

Стартовий список (13 каналів: мультфільми, розвиток, музика — українською, плюс категорія
English з ліцензованими каналами Cocomelon/Peppa Pig/Super Simple Songs/PBS KIDS/Sesame Street)
складено за публічними добірками україномовного дитячого контенту — **я (Claude) не переглядав
особисто відео на кожному з каналів**, тільки перевірив, що канали реальні й активні. Обов'язково
переглянь список у `app/src/main/assets/whitelist.json` і сам застосунок перед тим, як віддати
планшет дитині:

- `PBS KIDS` і `Sesame Street` додані за посиланням-здогадкою (`handle`) — якщо в застосунку канал
  не показує відео, зайди в Налаштування (іконка шестерні на головному екрані, потрібен PIN) і
  онови посилання, знайшовши актуальний `@handle` каналу на youtube.com.
- Прибрати чи додати канал можна прямо в застосунку (Налаштування → вставити посилання на канал
  youtube.com/@... або youtube.com/channel/UC...) — не обов'язково перезбирати APK.

## Батьківський контроль

- Перший запис PIN у діалозі — і стає постійним кодом батьків (захищає Налаштування і зняття
  з дитячого режиму).
- Іконка замка на головному екрані вмикає Android Screen Pinning — дитина не вийде на робочий
  стіл, поки не введено PIN.
- Ліміт часу перегляду на день вимикається/вмикається і налаштовується в Налаштуваннях.

## Обмеження першої версії

- Плеєр — офіційний YouTube IFrame Player у WebView (відповідає умовам використання YouTube).
- Квота YouTube Data API v3 безкоштовна — 10 000 одиниць/день; кожне відкриття каналу вперше за
  добу коштує ~3 одиниці (запити `channels`, `playlistItems`, `videos`), далі береться з кешу.
- Лічильник часу перегляду рахує, поки відкритий екран плеєра (наближено, без точного стеження за
  play/pause у самому відео).
